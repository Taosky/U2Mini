# U2Mini
-YouTube Web Extension. View the subscription video in the pop-up window.

-Youtube浏览器扩展（目前仅支持Firefox）.用于快速浏览订阅视频。

## 安装

https://addons.mozilla.org/zh-CN/firefox/addon/u2mini/
